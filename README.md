# 🎵 VibeCheck
**VibeCheck** is a high-energy, neon-themed music discovery engine. It allows users to break out of their musical bubbles by fetching random 30-second previews from the iTunes catalog.

### ✨ Features
* **Dynamic Discovery:** Search by Artist, Genre, or "Feeling Lucky" mode.
* **Vinyl Animation:** Album art turns into a spinning record during playback.
* **Smart Memory:** Remembers your volume levels and session history.
* **Responsive Design:** Optimized for both desktop and mobile.

### 🛠️ Built With
* **HTML5 & CSS3** (Glassmorphism UI)
* **Vanilla JavaScript** (ES6+)
* **iTunes Search API**
